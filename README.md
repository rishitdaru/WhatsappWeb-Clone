# WhatsappWeb-Clone
 - Cloned the WhatsApp Web Application using the MERN Stack

 - Technologies Used: 
      Frontend: React.js & Material-UI 
      Backend: Node.js & Express.js 
      Database: MongoDB 
      Real Time Chatting: socket.io
